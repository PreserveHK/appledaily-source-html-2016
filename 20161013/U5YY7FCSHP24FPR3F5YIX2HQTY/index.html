Note 7可退款或換S7 edge
退機要有原盒　用戶斥三星刁難

【本報訊】開售僅53日的短命手機Galaxy Note 7，前日遭南韓三星宣佈全球永久停產停售後，內地及台灣即安排退款或換機，惟獨香港慢半拍，用戶苦等一天，三星香港昨始公佈退機詳情，表示用戶可全數退款或換機。然而魔鬼卻在細節中，退款其中一項要求要有原廠包裝盒，引起用戶憤怒，質疑存心刁難，消委會促請三星香港顧及客戶不同需要，彈性處理。
記者：簡明恩　劉永明

三星香港公告指，已於香港和澳門購買Galaxy Note 7的用戶，可由昨天至12月31日期間，帶同Galaxy Note 7、原廠包裝盒和配件，以及收據正本到原購買地點（包括電訊商、特約零售商或經銷商），作全數退款或更換Galaxy S7 edge。

消委會促彈性處理

倘選擇更換全新Galaxy S7 edge，顧客將可取回兩者的差價，與Galaxy Note 7同時購買的原廠配件亦可進行全數退款或更換，另可選擇額外禮品，包括值398元的Samsung無線充電座，或128GB micro SD卡。用戶如有疑問，可致電客服熱線（3698 4698）。
此外，繼衛訊率先公佈退款安排後，3香港、數碼通、中國移動香港及香港電訊旗下csl昨天亦先後公佈相關安排，多數於12月31日前，帶同Note 7、單據及原廠配件到門市辦理。
Note 7被判死刑，嚇煞三星迷，昨晨旺角的三星客戶服務中心未開門，已擠滿人龍，而率先宣佈退款細節的衛訊更人頭湧湧，麥氏父女一早帶同3部Note 7到衛訊退機，分別換成Galaxy S7、iPhone及一部退款。她指，3部Note 7是其父、丈夫及家姑使用，於上月2日購入，為香港首批用戶，但傳出電池爆炸以來，一直憂心忡忡，害怕通宵開機及充電，「有得換梗係換」。麥斥責，三星安排退款混亂，應該一早公佈細節。

另有用戶批評，三星香港要用戶退款帶備原廠包裝盒，對此怒問：「邊度有人會拆機後仲留番個盒，屋企好大呀？」他稱每部手機均有編號、客戶單據及電腦紀錄，只要證明是Galaxy Note 7就應退款，毋須一定要有原廠盒。
消委會發言人表示，三星應予以彈性和合理的處理，照顧客戶的需要和情況，並敦促三星不論在客戶服務和各銷售點都須全面做好所有支援工作，確保回收能順暢及有序地執行。
至昨天下午5時為止，消委會收到共186宗有關Note 7的投訴，相關查詢有254宗。

南韓三星股價再跌

南韓三星電子昨大幅調低今年第三季盈利預測逾三成，料經營利潤將由7.8萬億韓圜（約537億港元）降至5.2萬億韓圜（358億港元）；收入則由49萬億韓圜降至47萬億韓圜。三星電子股價連跌三日，全日跌0.65%。
